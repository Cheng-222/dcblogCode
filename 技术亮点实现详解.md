## 3. 导航系统与路由实现

### 3.1 Vue Router的集成与配置
**实现方式：**
在`src/router/index.js`中配置路由：

```javascript
import { createRouter, createWebHistory } from 'vue-router'

const routes = [
  {
    path: '/',
    name: 'Home',
    component: () => import('../views/HomeView.vue')
  },
  {
    path: '/about',
    name: 'About',
    component: () => import('../views/AboutView.vue')
  },
  // 其他路由配置...
]

const router = createRouter({
  history: createWebHistory(),
  routes,
  scrollBehavior(to, from, savedPosition) {
    if (savedPosition) {
      return savedPosition
    }
    return { top: 0 }
  }
})

export default router
```

**技术亮点：**
- 使用路由懒加载(`() => import()`)减少初始加载体积
- 配置滚动行为，实现页面切换时的平滑滚动效果
- 采用HTML5 History模式，提供更干净的URL

### 3.2 响应式导航栏实现
**实现方式：**
在`src/components/NavBar.vue`中：

```vue
<script>
export default {
  data() {
    return {
      isScrolled: false,
      isMenuOpen: false
    }
  },
  mounted() {
    // 监听滚动事件
    window.addEventListener('scroll', this.handleScroll)
  },
  unmounted() {
    window.removeEventListener('scroll', this.handleScroll)
  },
  methods: {
    handleScroll() {
      this.isScrolled = window.scrollY > 50
    },
    toggleMenu() {
      this.isMenuOpen = !this.isMenuOpen
    },
    closeMenu() {
      this.isMenuOpen = false
    }
  }
}
</script>
```

**技术亮点：**
- 实现滚动监听，当页面滚动时动态改变导航栏样式
- 移动端使用汉堡菜单和动画过渡效果
- 点击导航链接自动关闭菜单，提升用户体验

## 4. 动态交互与动画效果

### 4.1 打字机效果实现
**实现方式：**
在`HomeView.vue`中实现打字机效果：

```vue
<script>
export default {
  data() {
    return {
      typedText: '',
      text: 'Web Developer & Designer',
      index: 0,
      speed: 100
    }
  },
  mounted() {
    this.typeWriter()
  },
  methods: {
    typeWriter() {
      if (this.index < this.text.length) {
        this.typedText += this.text.charAt(this.index)
        this.index++
        setTimeout(() => this.typeWriter(), this.speed)
      } else {
        // 完成后重置，可以实现循环效果
        setTimeout(() => {
          this.typedText = ''
          this.index = 0
          this.typeWriter()
        }, 2000)
      }
    }
  }
}
</script>
```

## 5. 项目展示与过滤系统

### 5.1 项目分类过滤功能
**实现方式：**
在`ProjectsView.vue`中：

```vue
<script>
export default {
  data() {
    return {
      activeCategory: 'all',
      projects: [
        { id: 1, title: '电子商务网站', category: 'web' },
        { id: 2, title: '移动应用设计', category: 'mobile' },

  computed: {
    filteredProjects() {
      if (this.activeCategory === 'all') {
        return this.projects
      }
      return this.projects.filter(project => project.category === this.activeCategory)
    }
  },
  methods: {
    filterProjects(category) {
      this.activeCategory = category
    }
  }
}
</script>
```

**技术亮点：**

## 6. 博客系统实现

### 6.1 博客搜索与分页
**实现方式：**
在`BlogView.vue`中：

```vue
<script>
export default {
  data() {
    return {
      searchQuery: '',
      activeCategory: 'all',
      posts: [
        // 博客文章数据...
      ],
      currentPage: 1,
      postsPerPage: 6
    }
  },
  computed: {
    filteredPosts() {
      let filtered = this.posts
      
      // 搜索过滤
      if (this.searchQuery) {
        const query = this.searchQuery.toLowerCase()
        filtered = filtered.filter(post => 
          post.title.toLowerCase().includes(query) ||
          post.content.toLowerCase().includes(query)
        )
      }
      
      // 分类过滤
      if (this.activeCategory !== 'all') {
        filtered = filtered.filter(post => post.category === this.activeCategory)
      }
      
      return filtered
    },
    paginatedPosts() {
      const startIndex = (this.currentPage - 1) * this.postsPerPage
      return this.filteredPosts.slice(startIndex, startIndex + this.postsPerPage)
    },
    totalPages() {
      return Math.ceil(this.filteredPosts.length / this.postsPerPage)
    }
  },
  methods: {
    // 翻页和搜索方法...
  }
}
</script>
```

**技术亮点：**
- 实现实时搜索功能，无需刷新页面
- 支持按分类筛选博客文章
- 实现客户端分页系统，优化大数据量显示

## 7. 表单交互与验证

### 7.1 联系表单实现
**实现方式：**
在`ContactView.vue`中：

```vue
<script>
export default {
  data() {
    return {
      form: {
        name: '',
        email: '',
        message: ''
      },
      errors: {},
      isSubmitting: false,
      submitMessage: ''
    }
  },
  methods: {
    validateForm() {
      this.errors = {}
      
      if (!this.form.name) {
        this.errors.name = '请输入您的姓名'
      }
      
      // 邮箱验证
      const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/
      if (!this.form.email || !emailRegex.test(this.form.email)) {
        this.errors.email = '请输入有效的邮箱地址'
      }
      
      if (!this.form.message) {
        this.errors.message = '请输入您的留言'
      }
      
      return Object.keys(this.errors).length === 0
    },
    async submitForm() {
      if (this.validateForm()) {
        this.isSubmitting = true
        
        try {
          // 这里是表单提交逻辑
          // 模拟API请求
          await new Promise(resolve => setTimeout(resolve, 1500))
          
          this.submitMessage = '您的留言已成功发送！'
          this.form = { name: '', email: '', message: '' }
          
          // 3秒后清除消息
          setTimeout(() => {
            this.submitMessage = ''
          }, 3000)
        } catch (error) {
          this.submitMessage = '发送失败，请稍后再试'
        } finally {
          this.isSubmitting = false
        }
      }
    }
  }
}
</script>
```

**技术亮点：**
- 实现客户端表单验证，包括邮箱格式验证
- 提供即时错误反馈和成功消息
- 使用async/await处理异步表单提交
- 实现表单状态管理（提交中、成功、失败）

## 总结

本个人网站基于Vue3技术栈，通过现代化的前端技术实现了多项亮点功能：

1. **模块化架构**：使用Vue3的组件化思想，实现代码的高度复用和维护性
2. **响应式设计**：通过CSS变量和媒体查询，确保在各种设备上的良好显示效果
3. **动态交互**：实现了多种动画效果，包括打字机效果、滚动触发动画等
4. **性能优化**：采用代码分割、懒加载等技术提升网站性能
5. **用户体验**：注重交互细节，如表单验证、分类过滤、平滑滚动等

网站整体采用黑色主题，搭配蓝色和粉色作为强调色，展现出专业而独特的视觉效果，同时通过精心设计的交互提升了用户体验。
